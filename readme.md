`images2rHTML.py` is a Python CLI tool that compiles multiple images from a folder into a single **Restricted Viewer HTML** file, complete with a navigable Outline (Table of Contents).

The generated HTML can be viewed directly in any modern web browser. It deters common actions like text selection, copying, right-clicking, image dragging, and printing. It does not modify your original images and operates completely offline without communicating with any external servers. Furthermore, by using a "Recovery password" set during creation, you can later import the HTML back into [MD//WORKS](https://github.com/HKJPN/markdown-editor) to restore the original Markdown, edit headings, add/remove images, or append text.
